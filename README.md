- [MIT Battery Degradation Dataset – Kaggle](https://www.kaggle.com/datasets/rickandjoe/mit-battery-degradation-dataset)
- [Battery Aging Dataset (Result Data v2) – Kaggle](https://www.kaggle.com/datasets/matthiasluh/battery-aging-dataset-result-data-v2)
- [NASA Battery Dataset – Kaggle](https://www.kaggle.com/datasets/patrickfleith/nasa-battery-dataset)
- [CALCE Battery Data (UMD)](https://calce.umd.edu/battery-data)
